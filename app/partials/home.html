Welcome!

This is a demonstration application designed for the WealthTab Team as part of a developer test. The goal is to create an AngularJs app that interracts with a node.js server in turns updating and requesting a local mongoDB server containing 2 collections (user and products). The API utilizes JSON Web Tokens to authenticate the user.
